🚀 Magento 2 Connector for Odoo 19

Simple yet Powerful. Multiple authentication methods. Clean ERP Sync.

This app connects Magento 2 and Odoo 19 using Magento's extensive API capabilities. It provides a secure, upgrade-safe, and transparent integration for synchronizing the most important data between the two platforms. With support for both OAuth 1.0a and Admin Token authentication, this connector offers flexible and secure access to the Magento API.

Built for businesses that want Magento 2 to work natively with Odoo, without fragile custom logic or upgrade risks.

🎯 What Makes This App Different?

Most Magento connectors focus on adding features. This connector focuses on doing the basics right - technically and functionally.

  • Odoo remains the single source of truth
  • Multiple authentication methods support
  • Multi-store support by filtering data based on store view
  • No hidden background processes
  • Handles all Magento product types - simple, configurable, bundle, grouped, virtual, downloadable

Just a transparent, predictable, and upgrade-safe integration.

🔁 Core Functional Capabilities

📦 Product Synchronization

  • Import products with variants
  • Variant mapping
  • SKUs synchronization

🛒 Order Management

  • Import Magento orders as Odoo Sales Orders
  • Automatic customer creation and mapping
  • Support for discounts, shipping charges, and taxes
  • Clear linkage between Magento and Odoo records

🚚 Delivery Synchronization

  • Auto delivery creation in Odoo when fulfilled on Magento
  • Auto creation & mapping of Magento delivery carriers
  • Push delivery along with tracking & carrier details from Odoo to Magento
  • Supports partial and multiple deliveries

📊 Inventory Synchronization

  • Stock updates from Odoo to Magento
  • Location-based inventory mapping
  • Controlled sync logic to prevent overselling
  • Fully compatible with standard Odoo Inventory

📍 Location Synchronization

  • Import Magento sources into Odoo
  • Support mapping of Magento sources with Odoo stock locations
  • Location-based order & stock processing
  • Multi-location support with accurate inventory sync

💰 Accounting-Friendly Design

  • Clean order values for downstream accounting
  • Tax-ready structure aligned with Odoo Invoicing
  • No custom accounting hacks or forced workflows

⚙ Full Sync Control & Transparency

  • Manual synchronization for precise control
  • Scheduled synchronization for automation
  • Detailed logs for every sync operation
  • Easy troubleshooting without technical dependency

You always know what synced, when, and how.

🏗 Designed for Real-World Odoo Usage

  • D2C brands using Odoo as their ERP
  • Businesses running one or multiple Magento stores
  • Teams that prefer standard Odoo behavior
  • Partners who value upgrade-safe, maintainable integrations
