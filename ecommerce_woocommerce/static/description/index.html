🚀 WooCommerce Connector for Odoo 19

Stable Sync. Secure API. ERP-Centric Design.

This app connects WooCommerce and Odoo 19 using WooCommerce's official REST API, enabling a structured and reliable connection between your online store and ERP.

Built for businesses that want WooCommerce to work cleanly with Odoo, without fragile custom logic or tightly coupled workflows.

🎯 What Makes This App Different?

Most WooCommerce connectors try to handle everything. This connector focuses on getting core ERP flows right.

  • Odoo remains the system of record
  • No fragile webhook-only logic
  • No theme or checkout dependencies
  • Designed for long-term maintainability

Just a transparent, predictable, and upgrade-safe integration between WooCommerce and Odoo.

🔁 Core Functional Capabilities

📦 Product Synchronization

  • Import simple and variable products
  • Variant attribute mapping
  • SKU synchronization
  • Clean product structure aligned with Odoo Product models

🛒 Order Management

  • Import WooCommerce orders as Odoo Sales Orders
  • Automatic customer creation and mapping
  • Support for coupons, shipping charges, and taxes
  • Sales orders ready for Odoo invoicing workflows

🚚 Inventory Synchronization

  • Stock updates from Odoo to WooCommerce
  • Scheduled synchronization via Odoo cron jobs
  • Manual sync available for controlled operations (debug mode)
  • Prevents stock mismatches between ERP and storefront

🏢 Multi-store Support

  • Support for multiple WooCommerce stores
  • Store-wise configuration in Odoo
  • Independent synchronization control per store

💰 Accounting-Friendly Design

  • Sales orders structured for clean downstream accounting
  • Compatible with standard Odoo Invoicing flows
  • No forced accounting customizations

⚙ Full Sync Control & Transparency

  • Scheduled synchronization for automation
  • Manual synchronization for testing and debugging
  • Detailed logs for every sync operation
  • Clear visibility into integration behavior

You always know what synced, when, and why.

🏗 Designed for Real-World Odoo Usage

  • D2C brands using WooCommerce as storefront
  • Businesses running Odoo as core ERP
  • Single or multi-store WooCommerce setups
  • Partners who value maintainable integrations
