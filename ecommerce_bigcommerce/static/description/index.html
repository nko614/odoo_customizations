🚀 BigCommerce Connector for Odoo 19

Reliable Sync. Clear Control. ERP-First Integration.

This app connects BigCommerce and Odoo 19 to help businesses manage products, orders, inventory, and deliveries directly from Odoo.

Built for teams that want predictable synchronization, clear operational control, and an integration that follows standard Odoo workflows without unnecessary complexity.

🎯 What Makes This App Different?

Most BigCommerce connectors try to automate everything. This connector focuses on clarity, control, and correctness.

  • Odoo remains the single source of truth
  • Explicit sync actions instead of hidden background processes
  • Clear separation between import and update operations
  • No forced workflows or hidden logic
  • Designed to be readable, debuggable, and upgrade-safe

Just a transparent, predictable, and maintainable integration.

🔁 Core Functional Capabilities

📦 Product Synchronization

  • Import products with variants
  • Variant mapping
  • SKUs synchronization

🛒 Order Management

  • Import BigCommerce orders as Odoo Sales Orders
  • Automatic customer creation and mapping
  • Support for discounts, shipping charges, and taxes
  • Clear linkage between BigCommerce and Odoo records

🚚 Delivery Synchronization

  • Auto delivery creation in Odoo when fulfilled on BigCommerce
  • Auto creation & mapping of BigCommerce delivery carriers
  • Push delivery along with tracking & carrier details from Odoo to BigCommerce
  • Supports partial and multiple deliveries

📊 Inventory Synchronization

  • Stock updates from Odoo to BigCommerce
  • Location-based inventory handling
  • Controlled sync logic to prevent overselling
  • Compatible with standard Odoo Inventory

📍 Location Synchronization

  • Import BigCommerce locations into Odoo
  • Location mapping with Odoo warehouses
  • Accurate stock updates per location
  • Multi-location support with accurate inventory sync

💰 Accounting-Friendly Design

  • Clean order values ready for invoicing
  • Tax-ready structure aligned with Odoo Invoicing
  • No custom accounting logic or forced workflows

⚙ Full Sync Control & Transparency

  • Manual synchronization for precise control
  • Scheduled synchronization for automation
  • Detailed logs for every sync operation
  • Easy troubleshooting without technical dependency

You always know what synced, when it synced, and how.

🏗 Designed for Real-World Odoo Usage

  • Businesses using Odoo as their primary ERP
  • Single or multiple BigCommerce stores
  • Teams that prefer standard Odoo behavior
  • Partners who value long-term maintainability

🧠 Design Philosophy

“A connector should respect both platforms — not fight them.”

This connector is built around standard BigCommerce behavior and native Odoo workflows to deliver an integration that is stable today and safe tomorrow.
