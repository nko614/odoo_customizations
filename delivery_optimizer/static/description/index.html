Installation

Custom/Addons Installation

  1. Download the module and place it in your Odoo custom/addons directory.
  2. Update the Odoo Apps list.
  3. Search for Delivery Route Optimizer in the Apps menu and click Install.
  4. Go to Settings > Delivery Route Optimizer to configure your subscription.

Odoo.sh Installation

  1. Click the Deploy on Odoo.sh button below to deploy this module to your Odoo.sh instance.
  2. Wait for the build to complete and the module to be installed.
  3. Go to Settings > Delivery Route Optimizer to configure your subscription.
Important: This app requires a valid subscription from Vikuno to leverage the Google Maps API and optimize routes. Without a subscription, route optimization features will not function. Please visit the Vikuno website to obtain your subscription ID.
